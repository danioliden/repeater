## 🌟 Overview
This app was developed to help students effectively manage their Quranic revision (murajaah). It generates personalized revision timetables based on the user's specific memorization progress.

## ✨ Key Features
* 📊 **Progress Tracking:** Monitor your memorization and revision milestones in real-time.
* 📚 **Interactive Notes:** Built-in resources to help users understand Quranic topics.
* 🔔 **Push Notifications:** Automated reminders based on your custom schedule.
